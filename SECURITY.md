# Security Policy

## Supported Versions

The table below indicates which versions of the project currently receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it through the official security contact channel:

* Contact: **[[markd.voznyuk@gmail.com](mailto:markd.voznyuk@gmail.com)]**
* Response time: You can expect an initial reply within **48 hours**.
* Process:

  * Your report will be reviewed.
  * If the issue is confirmed, it will be assigned for remediation.
  * After the fix is released, you will be notified.
  * If the issue is not considered a vulnerability, you will receive an explanation.

Thank you for helping keep the project secure.
